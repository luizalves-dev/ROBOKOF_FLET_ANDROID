# Template de novo layout — Robô KOF

Use esta pasta como ponto de partida para qualquer nova rede.

## Regras obrigatórias

- Não alterar parser de outra rede para cadastrar uma nova.
- Não descartar linhas silenciosamente.
- Não gerar TXT/fila antes do Excel de validação.
- Todo item precisa carregar origem: arquivo, aba/página, linha e regra aplicada.
- CNPJ sem matrícula deve ir para aba de cadastro, não sumir.
